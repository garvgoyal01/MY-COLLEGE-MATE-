# MY-COLLEGE-MATE-
College Companion is a modern, student-centric mobile application designed to simplify daily academic life for engineering college students. The app provides a unified platform for accessing essential college resources such as notices, societies, time tables, exam materials, internships, hackathons, and events, all in one place.

The goal of this project is to create a clean, fast, and engaging experience that helps students stay organized, prepared, and connected with their college ecosystem.
